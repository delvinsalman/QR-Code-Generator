## **Overview**  
This project is a **QR Code Generator**, where users can create QR codes with customizable features. The user can type or copy and paste a link into the search bar to generate a QR code. Additionally, users have the option to change the **color** of the QR code, adjust the **lines** within the QR code, and modify the **error correction level**, allowing the QR code to be tailored to their preferences.

## **🔧 Technical Aspects**  
This project was developed using **HTML**, **CSS**, and **JavaScript**. It uses various JavaScript libraries for QR generation and other customization features.

## **🚀 How to Use**  
1. **Input Link**:  
   - Type or copy and paste a URL into the input field.  
2. **Customize the QR Code**:  
   - Change the **color** and **lines** of the QR code.  
   - Adjust the **error correction level** for different types of scanning resilience.  
3. **Generate QR Code**:  
   - Click the button to generate your custom QR code.  
4. **Download QR Code**:  
   - Save the generated QR code by clicking the download button.
